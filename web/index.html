先用本地静态服务打开，再替换真实 JSON。

浏览器直接双击 HTML 通常不能安全读取同目录 JSON。最稳的方式是在 rqalpha 根目录启动一个本地静态服务。

  1. 静态演示模式：python3 -m http.server 8000，页面读取 web/result.json
  2. 可执行回测模式：rqalpha web --host 127.0.0.1 --port 8000
  3. 打开页面：http://127.0.0.1:8000/web/index.html
  4. 在可执行回测模式下，点击页面顶部“运行内置示例策略”即可直接触发本地 RQAlpha
